Side-by-Side Comparison

❌ Insecure Authentication

# Plaintext comparison — not secure
if username in users and
   users[username]['password_plain'] == password:
    success = True
  • Passwords stored in plaintext
  • Vulnerable to data leaks
  • Violates OWASP best practices

✅ Secure Authentication

# Hashed comparison — secure
if username in users and
   bcrypt.checkpw(password,
   users[username]['password_hashed']):
    success = True
  • Passwords hashed with bcrypt
  • Follows OWASP/NIST guidelines
  • Prevents reverse-engineering
